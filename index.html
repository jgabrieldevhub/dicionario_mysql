O que é o MySQL?

É o sistema de gestão de base de dados relacional mais utilizado no mundo. Imagine-o como um arquivo digital inteligente que organiza dados em tabelas relacionadas.

O Servidor Local

O que é o XAMPP?

O XAMPP é um pacote que instala tudo o que precisas para rodar um servidor no teu PC. Ele "acende as luzes" da tua base de dados localmente para poderes testar sem internet.

A Interface

O que é o DBeaver?

É a ferramenta visual para gerires os teus dados. Em vez de escreveres tudo na consola, o DBeaver dá-te uma interface profissional para veres tabelas, diagramas e editar registos com cliques.
